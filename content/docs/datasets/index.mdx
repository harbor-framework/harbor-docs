A [Harbor task](/docs/tasks) is an instruction, container environment, and test script. A dataset is a collection of tasks for evals and training.

Tasks can belong to multiple datasets, so datasets are a good way to create targeted eval groups.
You can publish public datasets or private datasets; private datasets are only available to members of your organization.

There are two ways to use datasets:

1. **Local datasets**: pass a local directory of tasks.
2. **Published datasets**: use Harbor package references like `org/name@tag` (for example, `my-org/my-dataset@1.0`).

## Local datasets

A local dataset runs directly from disk:

```bash
harbor run -p "<path/to/dataset>" -a "<agent>" -m "<model>"
```

## Published datasets

Run a published dataset with `-d`:

```bash
harbor run -d "my-org/my-dataset@1.0" -a "<agent>" -m "<model>"
```
